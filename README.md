

## Build Setup:

``` bash
# Скачиваем репозиторий
git clone https://github.com/Doctor-Hao/Testovoe.git

# Переходим в папку проекта
cd Testovoe 

# Install dependencies:
npm install

# Server with hot reload at http://localhost:8081/
npm run dev

# Output will be at dist/ folder
npm run build
```
